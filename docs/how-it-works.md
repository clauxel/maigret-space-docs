# How It Works - Maigret AI

Maigret AI turns username checks, recursive leads, tags, parsing, and multi-format reports into a managed workflow that people can review and hand off.

## Enter the lead

Start with one username, profile URL, or small alias set and define the investigation purpose.

## Search and parse

Use tags, site coverage, profile parsing, and recursive leads to build evidence.

## Review matches

Separate likely matches from false positives and note why each hit matters.

## Export the case

Create the format the next human or system needs: graph, HTML, PDF, CSV, JSON, or Telegram handoff.
