# Guides - Maigret AI

Maigret AI turns username checks, recursive leads, tags, parsing, and multi-format reports into a managed workflow that people can review and hand off.

## Start with a narrow scope

Define purpose, target handles, allowed sources, report format, and stop conditions before searching broadly.

## Review false positives

Check profile content, usernames, avatars, links, dates, and platform context before treating a hit as evidence.

## Use the upstream repo

Inspect Maigret documentation, supported identifiers, install routes, and licenses when you need local ownership.

## Pricing and Support

Plans cover managed searches, report artifacts, caseboards, export formats, and team handoff workflows.
