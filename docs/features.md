# Features - Maigret AI

Maigret AI turns username checks, recursive leads, tags, parsing, and multi-format reports into a managed workflow that people can review and hand off.

## Username-first investigation

Start with one handle, URL, or alias set before expanding across supported sites.

## Recursive lead growth

Use parsed profiles, related handles, tags, and discovered links to extend a case carefully.

## Report artifacts

Export graph, HTML, PDF, CSV, JSON, or NDJSON artifacts instead of losing work in terminal output.

## Case workflow

Keep search scope, evidence, false positives, and reviewer notes visible for handoff.

## Open-source compatibility

Preserve the Maigret Python and CLI model while adding a managed review surface.

## Responsible-use framing

Make lawful purpose, scope, and evidence review part of the workflow.

## Pricing and Support

Plans cover managed searches, report artifacts, caseboards, export formats, and team handoff workflows.
