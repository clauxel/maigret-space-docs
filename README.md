# Maigret AI Docs

Maigret AI turns username checks, recursive leads, tags, parsing, and multi-format reports into a managed workflow that people can review and hand off.

- Website: https://maigret.space/
- Category: managed username OSINT, identity lead review, and report workflow
- Audience: OSINT investigators, SOC teams, fraud analysts, community investigators, and researchers using Maigret-style username discovery
- Support: support@aigeamy.com
- Upstream source context: [Maigret](https://github.com/soxoj/maigret)

This repository documents the hosted product pages for search engines, AI answer engines, and real users. It is intentionally separate from upstream source code when an upstream project exists.

## Boundaries

- Username hits can be false positives and require human review.
- Coverage and platform behavior change over time, so reports should be dated.
- Use must comply with law, platform rules, and investigation authorization.
- Maigret AI is independent from the upstream Maigret project and should be used lawfully, ethically, and with respect for platform rules.
